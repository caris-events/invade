# 笨他侵略

我們的主旨並非讓人「仇恨中國」或是無腦地「抵制使用中國產品」，但我們希望你知道每個人的所作所為。這個網路資料庫記載了與中共政府有關的團體、服務還有侵略行為（例如：言論審查、侵害人權）。

你可以透過 [Google 表單](https://invade.tw/contribute)來建議新增、編輯條目。
